# 太阳系比例模型课堂海报

## 中文说明

制作一张明亮清晰的太阳系课堂海报，标明比例尺距离和行星名称。

## 使用场景

* 解释概念、设计练习和学习计划
* 生成课程材料、测验或教学活动
* 把复杂知识转化为清晰的学习路径
* 围绕 Art、Science、visual 等主题生成结构化结果

## 适用人群

* 教师
* 学生
* 培训人员

## 中文 Prompt

```md
设计一张课堂海报，用行星之间的比例距离来说明太阳系。海报应该明亮、清晰、信息丰富，包括每个行星的名称。这张海报旨在用于教育目的，帮助学生了解太阳系的结构和规模。

输出要求：
- 按原 prompt 要求的格式输出。
```
