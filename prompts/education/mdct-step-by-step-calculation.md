# MDCT逐步计算

## 中文说明

该提示逐步执行MDCT以进行多媒体压缩分配。

## 使用场景

* 解释概念、设计练习和学习计划
* 生成课程材料、测验或教学活动
* 把复杂知识转化为清晰的学习路径
* 适合复制给图像生成模型，控制主体、构图、风格和画面细节。

## 适用人群

* 教师
* 学生
* 培训人员

## 中文 Prompt

```md
对输入序列实施MDCT：

x(n) = [1, 2, 3, 4]

步骤：
1. 识别N和2N
2.应用MDCT公式
3. 清晰显示余弦值
4. 显示分步计算表
5.给出最终系数

输出要求：
- 按原 prompt 要求的格式输出。
```
