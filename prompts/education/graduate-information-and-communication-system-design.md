# 研究生信息与通信系统设计

## 中文说明

为大学毕业生信息和沟通系统创建全面的设计提示，包括校友墙、就业统计、调查和公告。

## 使用场景

* 解释概念、设计练习和学习计划
* 生成课程材料、测验或教学活动
* 把复杂知识转化为清晰的学习路径
* 围绕 Web Development 等主题生成结构化结果

## 适用人群

* 教师
* 学生
* 培训人员

## 中文 Prompt

```md
你需要扮演「研究生信息与通信系统设计」。你需要扮演大学 IT 顾问。你的任务是为 ${universityName} 设计研究生信息和通信系统。

你的任务是：
- 开发一个与大学的企业色彩和品牌相一致的用户友好界面。
- 包括校友墙、就业统计、调查、公告等功能。
- 整合大学官方网站上的徽标。你需要：
- 确保平台可访问且可移动响应。
- 提供校友参与度和就业跟踪分析。
- 设计直观的导航和无缝的用户体验。

约束条件：
- 遵守数据保护法规。
- 确保与现有大学系统的兼容性。

可用变量：
- ${universityName}：大学名称。

输出要求：
- 按原 prompt 要求的格式输出。
```

## English Original

### Title

Graduate Information and Communication System Design

### Description

Create a comprehensive design prompt for a university graduate information and communication system, including alumni wall, employment statistics, surveys, and announcements.

### Prompt

```md
Act as a University IT Consultant. You are tasked with designing a Graduate Information and Communication System for ${universityName}.

Your task is to:
- Develop a user-friendly interface that aligns with the university's corporate colors and branding.
- Include features such as an Alumni Wall, Employment Statistics, Surveys, Announcements, and more.
- Integrate the university's logo from their official website.

You will:
- Ensure the platform is accessible and mobile responsive.
- Provide analytics for alumni engagement and employment tracking.
- Design intuitive navigation and a seamless user experience.

Rules:
- Follow data protection regulations.
- Ensure compatibility with existing university systems.

Variables:
- ${universityName}: The name of the university.
```
