# 逐页构建

## 中文说明

此提示将帮助你在启动提示后逐页构建网站

## 使用场景

* 生成 UI/UX、视觉风格和设计规范
* 分析界面问题并提出改进方向
* 把需求转化为可执行的设计说明
* 围绕 design、ui-ux 等主题生成结构化结果

## 适用人群

* 设计师
* 产品经理
* 前端开发者

## 中文 Prompt

```md
根据批准的概念，构建[主页/关于/等]页面。限制条件：
- 带 Tailwind 的单文件 React 组件
- 移动优先，响应迅速
- 性能预算：除非有合理理由，否则任何库都不会超过 50kb
- [第一阶段的特定互动]一定是英雄时刻
- 使用前端设计技巧来提高设计质量

显示该组件。在进入下一页之前我会先回顾一下。

约束条件：
- 严格保留原 prompt 中的限制条件、禁止事项和输出边界。
```

## English Original

### Title

Page-by-Page Build

### Description

This prompt will help you to build your web site page by page after the kick starter prompt

### Prompt

```md
Based on the approved concept, build the [Homepage/About/etc.] page.

Constraints:
- Single-file React component with Tailwind
- Mobile-first, responsive
- Performance budget: no library over 50kb unless justified
- [Specific interaction from Phase 1] must be the hero moment
- Use the frontend-design skill for design quality

Show me the component. I'll review before moving to the next page.
```
