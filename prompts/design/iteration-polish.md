# 迭代与完善

## 中文说明

此提示将帮助你完善和完善设计迭代。

## 使用场景

* 生成 UI/UX、视觉风格和设计规范
* 分析界面问题并提出改进方向
* 把需求转化为可执行的设计说明
* 围绕 ui-ux、design 等主题生成结构化结果

## 适用人群

* 设计师
* 产品经理
* 前端开发者

## 中文 Prompt

```md
根据以下标准检查当前的 ${page}：
- 英雄部分是否能在 3 秒内产生清晰的情绪React？
- 每个断点处的版式层次结构是否清晰？
- 互动是有目的的还是装饰性的？
- 这在质量上感觉像 ${reference_site_x} 但在身份上却截然不同吗？通过推理提出 3 项具体改进建议，然后实施。
```

---

## English Original

### Title

Iteration & Polish

### Description

This prompt will help you to refine and polish over the design iteration.

### Prompt

```md
Review the current ${page} against these criteria:
- Does the hero section create a clear emotional reaction in <3 seconds?
- Is the typography hierarchy clear at every breakpoint?
- Are interactions purposeful or decorative?
- Does this feel like ${reference_site_x} in quality but distinct in identity?

Suggest 3 specific improvements with reasoning, then implement them.
```
