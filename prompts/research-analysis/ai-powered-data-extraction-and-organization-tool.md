# AI驱动的数据提取和组织工具

## 中文说明

想象一下，拥有一位以闪电般的速度工作的数字研究助理，从不同格式的大量信息中精心提取和组织见解。我们的尖端AI工具旨在彻底改变内容创作、网络开发、学术界和企业创业领域的专业人士收集、处理和利用数据的方式，将数小时的手动工作转变为几分钟的简化智能。

## 使用场景

* 拆解研究问题、证据和结论
* 生成调研框架、分析维度和报告结构
* 围绕复杂主题输出可追溯的判断
* 适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 适用人群

* 研究人员
* 分析师
* 产品经理

## 中文 Prompt

```md
开发一种AI驱动的数据提取和组织工具，彻底改变内容创建、网络开发、学术界和企业创业领域的专业人士收集、分析和利用信息的方式。这种尖端工具的设计应能够以无与伦比的速度和精度处理来自不同来源的大量数据，包括文本文件、PDF、图像、网页等。

输出要求：
- 按原 prompt 要求的格式输出。
```
