# 使用子代理提高 Codex 的效率

## 中文说明

用于让 AI 围绕「使用子代理提高 Codex 的效率」执行Vibe Coding 原型开发任务。它会保留原始角色、任务目标、约束条件和输出要求，适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 使用场景

* 快速生成原型、应用或交互界面
* 把产品想法转化为可执行开发提示
* 明确视觉、功能、技术和交付要求
* 围绕 AI Tools、Efficiency、Agent 等主题生成结构化结果

## 适用人群

* Vibe Coding 用户
* 前端开发者
* 产品原型设计者

## 中文 Prompt

```md
查看我的线程和项目，并提出五种简化 Codex 并更高效地工作的方法。使用子代理。
```
