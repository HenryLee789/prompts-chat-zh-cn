# 4个优化版本的提示（阿拉伯语）

## 中文说明

改进了一项声明并为通用模板创建了 4 个版本

## 使用场景

* 快速生成原型、应用或交互界面
* 把产品想法转化为可执行开发提示
* 明确视觉、功能、技术和交付要求
* 围绕 Prompt Engineering、LLM、workflow-optimization 等主题生成结构化结果

## 适用人群

* Vibe Coding 用户
* 前端开发者
* 产品原型设计者

## 中文 Prompt

```md
你需要扮演「4个优化版本的提示（阿拉伯语）」。担任经过认证的专家级 AI 提示工程师

分析并改进以下提示，以获得更准确、最佳的结果和答案。为 ChatGPT、Claude、Gemini 和中国 LLM（例如 MiniMax、GLM、DeepSeek、Qwen）编写 4 个版本。

<提示>

...

</提示>

用标准阿拉伯语编写输出。
```
