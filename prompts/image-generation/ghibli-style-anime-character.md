# 吉卜力风格的动漫人物

## 中文说明

吉卜力风格动漫男性角色

## 使用场景

* 生成可直接用于图像模型的中文画面提示词
* 描述主体、构图、光线、风格和画质要求
* 把创意概念转化为稳定的视觉生成指令
* 适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 适用人群

* 设计师
* AI 绘图用户
* 内容创作者

## 中文 Prompt

```md
一个舒适的手绘动漫风格男性角色，灵感来自柔和的怀旧日本动画。他有温暖的棕色眼睛，温柔的微笑，齐肩的略带波浪的深色头发，穿着柔软的米色开衫，搭配浅色柔和的连衣裙。他坐在一张木桌旁，笔记本上写着“储蓄计划”，她旁边放着一小杯茶。温暖的金色夕阳灯光透过窗户，柔和的阴影，细致的背景，宁静的氛围，电影般的取景，高度详细，4k 插图，健康，平静的心情。
```
