# 创建这个很酷的涂鸦。

## 中文说明

给用户的提示： [主题] = 整体氛围/故事。 [对象列表] = 你希望 AI 在人物周围绘制的特定涂鸦。 示例： 主题：赛博朋克科技 对象：电路线、全息图、数据流、发光箭头、技术图标、闪电。

## 使用场景

* 生成可直接用于图像模型的中文画面提示词
* 描述主体、构图、光线、风格和画质要求
* 把创意概念转化为稳定的视觉生成指令
* 围绕 image-generation、Nano Banana、ChatGPT 等主题生成结构化结果

## 适用人群

* 设计师
* AI 绘图用户
* 内容创作者

## 中文 Prompt

```md
使用上传的图像作为基本照片，不要更改人物的脸部、身体或姿势。在主题周围添加手绘的白色涂鸦插图，粗略的标记/粉笔风格，不完美的俏皮线条，涂鸦应该出现在照片的顶部。涂鸦主题：${theme} 包含以下元素：${object_list} 保持背景完整，不遮挡脸部，编辑美学海报风格，构图干净，质量高。
```

## English Original

### Title

Create This cool Doodles.

### Description

Tip for users:

[THEME] = the overall vibe/story.
[OBJECT LIST] = the specific doodles you want the AI to draw around the person.

Example:
Theme: Cyberpunk Tech
Objects: circuit lines, holograms, data streams, glowing arrows, tech icons, lightning bolts.

### Prompt

```md
Using the uploaded image as the base photo, do NOT change the person's face, body, or pose. Add hand-drawn white doodle illustrations around the subject, sketchy marker / chalk style, imperfect playful lines, doodles should appear drawn on top of the photo. Doodle theme: ${theme} Include elements such as: ${object_list} Keep the background intact, do not cover the face, editorial aesthetic poster style, clean composition, high quality.

```
