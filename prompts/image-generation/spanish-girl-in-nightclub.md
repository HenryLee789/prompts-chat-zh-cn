# 夜总会的西班牙女孩

## 中文说明

用于让 AI 围绕「夜总会的西班牙女孩」执行图像生成提示词设计任务。它会保留原始角色、任务目标、约束条件和输出要求，适合复制给图像生成模型，控制主体、构图、风格和画面细节。

## 使用场景

* 生成可直接用于图像模型的中文画面提示词
* 描述主体、构图、光线、风格和画质要求
* 把创意概念转化为稳定的视觉生成指令
* 适合复制给图像生成模型，控制主体、构图、风格和画面细节。

## 适用人群

* 设计师
* AI 绘图用户
* 内容创作者

## 中文 Prompt

```md
{
  “行动”：“图像生成”，
  "action_input": "一张全身照片，垂直格式 9:16 AR 的 Natalia，一位 23 岁的西班牙女性，有着深棕色的长卷发和绿色的眼睛。她身处一家拥挤、灯光昏暗、霓虹灯点缀下的当代罗马夜店。她穿着一件合身的极短黑色丝质吊带裙，深乳沟凸显了她的曲线和突出的胸部。脚上穿着高跟凉鞋。她看起来容光焕发，不羁，她手里拿着一杯酒，边跳舞边笑，周围是背景中模糊的人影，气氛朦胧、充满活力，充满电影感，捕捉到了狂野自由和感官超载的时刻。”
}
```
