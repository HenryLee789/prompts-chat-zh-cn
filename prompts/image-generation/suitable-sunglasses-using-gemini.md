# 双子座适合的太阳镜

## 中文说明

提供图片，双子座建议合适的太阳镜框

## 使用场景

* 生成可直接用于图像模型的中文画面提示词
* 描述主体、构图、光线、风格和画质要求
* 把创意概念转化为稳定的视觉生成指令
* 适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 适用人群

* 设计师
* AI 绘图用户
* 内容创作者

## 中文 Prompt

```md
使用上传图像提供图像，并在面部放置合适的太阳镜框

输出要求：
- 按原 prompt 要求的格式输出。
```
