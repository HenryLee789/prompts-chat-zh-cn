# 乐高电影风格提示词

## 中文说明

重新想象为乐高电影中的场景。

## 使用场景

* 生成可直接用于图像模型的中文画面提示词
* 描述主体、构图、光线、风格和画质要求
* 把创意概念转化为稳定的视觉生成指令
* 围绕 Movies、Art、Nano Banana 等主题生成结构化结果

## 适用人群

* 设计师
* AI 绘图用户
* 内容创作者

## 中文 Prompt

```md
{
  "TASK": "重新想象乐高电影中的场景。",
  "VISUAL_ID": "塑料砖的微距摄影。定格动画的感觉。",
  "CHARACTERS": "乐高小人仔。C 形手、圆柱形头、彩绘脸。",
  "SURFACE": "有光泽的塑料质感、指纹、塑料上的划痕。",
  "BG": "完全由乐高积木搭建。景深焦点。",
  “NEG”：“人体皮肤，布料纹理，逼真的解剖学，2D，绘画，卡通，动漫，柔软。”
}
```

Lego Movie Style Prompt
