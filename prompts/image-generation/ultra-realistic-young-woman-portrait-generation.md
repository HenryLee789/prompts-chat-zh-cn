# 超写实年轻女性肖像一代

## 中文说明

生成具有特定特征的年轻女性的超真实图像，包括白皙的皮肤、雀斑、蓝眼睛、长长的金发等等，以自然的抓拍照片风格拍摄。

## 使用场景

* 生成可直接用于图像模型的中文画面提示词
* 描述主体、构图、光线、风格和画质要求
* 把创意概念转化为稳定的视觉生成指令
* 围绕 AI Tools、Midjourney、DALL-E 等主题生成结构化结果

## 适用人群

* 设计师
* AI 绘图用户
* 内容创作者

## 中文 Prompt

```md
生成一个 22 岁年轻女性的超现实图像，具有以下特征：
- 皮肤白皙，有轻微雀斑
- 蓝眼睛，对称的脸
- 金色长直发，中分
- 自然粉嫩双唇，柔和自然妆容
- 身材苗条，脸型一致，外观一致
- 使用 iPhone 后置摄像头拍摄的照片
- 自然、不完美的肌肤纹理
- 真实的灯光，坦诚的照片风格

确保图像高度真实，捕捉抓拍照片的精髓和所有指定细节。
```

## English Original

### Title

Ultra-Realistic Young Woman Portrait Generation

### Description

Generate an ultra-realistic image of a young woman with specified features, including fair skin with freckles, blue eyes, long blonde hair, and more, captured in a natural candid photo style.

### Prompt

```md
Generate an ultra-realistic image of a young woman aged 22 years with the following features:
- Fair skin with light freckles
- Blue eyes, symmetrical face
- Long straight blonde hair, middle part
- Natural pink lips, soft natural makeup
- Slim body, same face, consistent appearance
- Photo captured using an iPhone back camera
- Natural, imperfect skin texture
- Realistic lighting, candid photo style

Ensure the image is high in realism, capturing the essence of a candid photo with all specified details.
```
