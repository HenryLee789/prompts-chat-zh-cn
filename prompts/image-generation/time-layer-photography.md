# 时间层摄影

## 中文说明

在单个图像中叠加同一位置的不同时区。不是早上/中午/晚上，而是相隔数年或几个世纪。

## 使用场景

* 生成可直接用于图像模型的中文画面提示词
* 描述主体、构图、光线、风格和画质要求
* 把创意概念转化为稳定的视觉生成指令
* 围绕 DALL-E、Midjourney 等主题生成结构化结果

## 适用人群

* 设计师
* AI 绘图用户
* 内容创作者

## 中文 Prompt

```md
${location:Galata Tower, Istanbul} 的单张照片，框架被分为有机、流动的部分，每个部分都显示不同的时代：${era1:1890s sepia Ottoman period}、${era2:1960s faded color}、${era3:present day digital clarity}。时代之间的过渡是无缝的，通过建筑细节、人们的服装和车辆融为一体。相同的拍摄角度，相同的视角，不同的时间相互渗透。街道层视图。逼真再现每个时代真实的摄影风格。
```

## English Original

### Title

Time Layer Photography

### Description

Overlaying different time zones of the same location in a single image. Not morning/noon/night, but years or centuries apart.

### Prompt

```md
A single photograph of ${location:Galata Tower, Istanbul} where the frame is divided into organic, flowing sections, each showing a different era: ${era1:1890s sepia Ottoman period}, ${era2:1960s faded color}, ${era3:present day digital clarity}. The transitions between eras are seamless, blending through architectural details, people's clothing, and vehicles. Same camera angle, same perspective, different times bleeding into each other. Street level view. Photorealistic for each era's authentic photography style.
```
