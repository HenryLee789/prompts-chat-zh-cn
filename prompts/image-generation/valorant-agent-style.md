# 勇敢特工风格

## 中文说明

《勇敢特工》艺术风格提示。

## 使用场景

* 生成可直接用于图像模型的中文画面提示词
* 描述主体、构图、光线、风格和画质要求
* 把创意概念转化为稳定的视觉生成指令
* 围绕 Nano Banana、Games、Art 等主题生成结构化结果

## 适用人群

* 设计师
* AI 绘图用户
* 内容创作者

## 中文 Prompt

```md
{ "TASK": "设计独特的《Valorant》特工关键艺术。Riot Games 艺术风格。",
"VISUAL_ID": "锐利的2.5D数字绘画。动漫与西方漫画的融合。哑光纹理，线条干净，无噪音。",
"PALETTE": "主要：深石板蓝 (#0f1923)。品牌：超红 (#ff4655)。能力：霓虹灯高光。",
“代理”：“运动、自信。未来科技街头服饰（肩带、风衣、战术手套）。锋利的面部平面。头发：浓密、雕刻的块状（无发丝）。”、“效果”：“挥舞风格化的元素力量（固体能量形式，而不是现实粒子）。”、“BG”：“抽象运动图形、平面几何平面、动态排版。红色/深色对比切片框架。”,
“LIGHT”：“强边缘照明，硬边投射阴影。”，“NEG”：“照片写实主义，砂砾，污垢，油画，柔焦，3D 渲染，闪亮的金属，凌乱，噪音，模糊。”
}//你可以在此处添加名称和技能或尺寸（例如 16:9）。
```
