# 瑞克和莫蒂

## 中文说明

将场景重新想象为“瑞克和莫蒂”电视节目截图

## 使用场景

* 生成可直接用于图像模型的中文画面提示词
* 描述主体、构图、光线、风格和画质要求
* 把创意概念转化为稳定的视觉生成指令
* 围绕 Art、Games、Nano Banana 等主题生成结构化结果

## 适用人群

* 设计师
* AI 绘图用户
* 内容创作者

## 中文 Prompt

```md
{
  "TASK": "将场景重新想象为《瑞克和莫蒂》电视节目截图。",
  "VISUAL_ID": "2D 矢量动画，成人游泳风格 (Justin Roiland)。平坦的颜色，均匀的细黑色轮廓。",
  “角色”：“将人类转化为‘瑞克和莫蒂’的解剖结构。管状/面条状四肢，下垂的姿势。眼睛：大的白色球体，带有独特的‘潦草’不规则黑色瞳孔（摇晃的点）。表情：冷漠、惊慌或流口水。”,
  "OUTFIT": "将复杂的战术装备简化为平面卡通科幻服装。消除纹理噪音；仅保留标志性形状。",
  "BG": "外星维度或凌乱的车库。摇摇欲坠的有机线条，奇怪的科幻纹理（洞，粘液）。调色板：霓虹门绿，柔和的大地色调，苍白的肤色。",
  "RENDER": "零梯度。平坦的照明。没有阴影或最小的硬卡通着色。干净的矢量外观。",
  “NEG”：“3D、逼真、体积照明、渐变、详细阴影、动画、噪音、绘画、模糊、勇敢风格、锐角。”
}
```
