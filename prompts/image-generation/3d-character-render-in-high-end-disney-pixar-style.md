# 高端迪士尼皮克斯风格的 3D 角色渲染

## 中文说明

此提示概述了基于所提供照片中的男子的高端皮克斯/迪士尼风格 3D 角色渲染的创建。它指定保留他准确的面部特征、发型、胡须和带有标志的条纹衬衫。该角色将全身站立在干净的白色工作室背景上，利用柔和、均匀的灯光和微妙的阴影来实现超清晰、4K 逼真的卡通效果。

## 使用场景

* 生成可直接用于图像模型的中文画面提示词
* 描述主体、构图、光线、风格和画质要求
* 把创意概念转化为稳定的视觉生成指令
* 围绕 Art、Nano Banana 等主题生成结构化结果

## 适用人群

* 设计师
* AI 绘图用户
* 内容创作者

## 中文 Prompt

```md
基于上传的照片，采用高端皮克斯迪士尼动画风格的 3D 角色渲染。保留面部结构、表情、发型和独特特征。可爱而真实的比例，干净的拓扑，光滑的皮肤，细致的眼睛。全身站立在纯白色的工作室背景上，柔和均匀的灯光，脚下微妙的自然阴影，全局照明，没有道具，没有干扰。超锐利、4K、高细节、基于物理的渲染、平衡的色彩、电影深度、专​​业工作室外观、对称取景、逼真的卡通效果。
```

## English Original

### Title

3D Character Render In High-End Disney Pixar Style

### Description

This prompt outlines the creation of a high-end Pixar/Disney style 3D character render, based on the man in the provided photo. It specifies preserving his exact facial features, hairstyle, beard, and striped shirt with the logo. The character is to be depicted standing full-body on a clean white studio background, utilizing soft, even lighting and a subtle shadow to achieve an ultra-sharp, 4K photoreal cartoon finish.

### Prompt

```md
3D character render in high-end Pixar Disney animation style, based on the uploaded photo. Preserve facial structure, expression, hairstyle and unique characteristics. Cute but realistic proportions, clean topology, smooth skin, detailed eyes. Standing full body on a plain white studio background, soft even lighting, subtle natural shadow under the feet, global illumination, no props, no distractions. Ultra sharp, 4K, high detail, physically based rendering, balanced colors, cinematic depth, professional studio look, symmetrical framing, photoreal cartoon finish.
```
