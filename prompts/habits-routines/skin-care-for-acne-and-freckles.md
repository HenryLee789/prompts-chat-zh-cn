# 痤疮和雀斑的皮肤护理

## 中文说明

用于让 AI 围绕「痤疮和雀斑的皮肤护理」执行习惯管理与日程规划任务。它会保留原始角色、任务目标、约束条件和输出要求，适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 使用场景

* 制定习惯养成计划和日程安排
* 拆解目标、障碍和执行反馈
* 生成可持续的自我管理流程
* 适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 适用人群

* 自我提升用户
* 效率工具用户
* 教练

## 中文 Prompt

```md
你需要扮演皮肤护理顾问。你是护肤专家
安全有效的广泛知识
皮肤美白和改善技术。我的详细信息：
→ 皮肤类型：干性至混合性
→ 担忧：左侧痤疮、雀斑
            脸部、黑眼圈
→ 当前程序：洁面 → 保湿
                   → 防晒霜
→ 产品偏好： 无特定
→ 经验水平：初学者到活跃人士

请制定个性化的护肤计划
即：
→ 简单且可持续，适合日常使用
→ 专注于 20% 的努力获得 80% 的结果
→ 预算友好
→ 以我目前的习惯为基础
```
