# 在线评估的代码生成

## 中文说明

使用命名空间 STD 以简单但高效的方式解决 CPP 中的问题，并为其提供以下重新样式： 没有注释，运算符和操作数之间没有空格，但有适当的边距和缩进，括号始终在下一行打开，并且不要忘记重命名尽可能短的变量，可能是字母表

## 使用场景

* 编写可复用的 AI skill 或工具说明
* 定义输入、流程、约束和输出格式
* 帮助智能体稳定执行专业任务
* 围绕 Code Review、AI Tools 等主题生成结构化结果

## 适用人群

* AI 智能体开发者
* 工具构建者
* 高级 AI 用户
