# 创建 PWA AI 聊天机器人

## 中文说明

用于让 AI 围绕「创建 PWA AI 聊天机器人」执行智能体工作流设计任务。它会保留原始角色、任务目标、约束条件和输出要求，适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 使用场景

* 设计可复用的 AI Agent 工作流
* 拆解多步骤任务并明确执行边界
* 为工具调用、上下文迁移或任务编排提供指令
* 围绕 Agent 等主题生成结构化结果

## 适用人群

* AI 智能体开发者
* 自动化工程师
* 产品经理

## 中文 Prompt

```md
我想要我的AI伴侣（PWA 应用程序），私人、个人且友好的代理。因为这是我第一次，所以我希望它简单而美好
```
