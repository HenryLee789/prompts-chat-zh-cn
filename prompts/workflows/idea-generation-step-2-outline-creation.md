# 第 2 步：创建大纲

## 中文说明

根据生成的想法创建结构化大纲

## 使用场景

* 拆解多阶段任务和执行顺序
* 定义输入、步骤、检查点和输出结果
* 把复杂工作转化为可复用流程
* 适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 适用人群

* 项目经理
* 自动化工程师
* 团队负责人

## 中文 Prompt

```md
根据上一步中产生的想法，创建详细的大纲。通过以下内容构建你的大纲：
- 主要部分和小节
- 涵盖的要点
- 每个部分的预计时间/精力
- 各部分之间的依赖关系

以清晰的层次结构格式化大纲。
```

## English Original

### Title

Step 2: Outline Creation

### Description

Create a structured outline from the generated ideas

### Prompt

```md
Based on the ideas generated in the previous step, create a detailed outline.

Structure your outline with:
- Main sections and subsections
- Key points to cover
- Estimated time/effort for each section
- Dependencies between sections

Format the outline in a clear, hierarchical structure.
```
