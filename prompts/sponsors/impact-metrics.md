# 影响指标

## 中文说明

用于让 AI 围绕「影响指标」执行开源赞助内容设计任务，并保留原 prompt 的角色、约束、执行步骤和输出格式。适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 使用场景

* 撰写开源项目赞助说明
* 设计赞助层级、项目故事和影响展示
* 提升 GitHub Sponsors 页面的可信度和转化率
* 围绕 Project、Metrics、Impact 等主题生成结构化结果

## 适用人群

* 开源维护者
* 开发者
* 项目运营者

## 中文 Prompt

```md
创建一个引人注目的数据驱动部分，显示[项目名称]的影响：下载、用户帮助、解决的问题和社区增长统计数据。
```
