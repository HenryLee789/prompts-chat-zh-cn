# 认可赞助商

## 中文说明

用于让 AI 围绕「认可赞助商」执行开源赞助内容设计任务，并保留原 prompt 的角色、约束、执行步骤和输出格式。适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 使用场景

* 撰写开源项目赞助说明
* 设计赞助层级、项目故事和影响展示
* 提升 GitHub Sponsors 页面的可信度和转化率
* 围绕 Community、Engagement、Recognition 等主题生成结构化结果

## 适用人群

* 开源维护者
* 开发者
* 项目运营者

## 中文 Prompt

```md
列出我可以认可或让赞助商参与我的项目社区的方式（例如，特殊的 Discord 角色、早期功能访问、私人问答会议）。
```
