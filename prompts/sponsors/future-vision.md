# 未来愿景

## 中文说明

打造引人入胜的简历

## 使用场景

* 撰写开源项目赞助说明
* 设计赞助层级、项目故事和影响展示
* 提升 GitHub Sponsors 页面的可信度和转化率
* 围绕 Bio、Vision、Future 等主题生成结构化结果

## 适用人群

* 开源维护者
* 开发者
* 项目运营者

## 中文 Prompt

```md
写一份引人注目的愿景声明，说明我认为[项目/工作]在未来 2-3 年内的发展方向，以及赞助商如何参与这一旅程。
```
