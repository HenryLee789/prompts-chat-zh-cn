# 显示直接影响

## 中文说明

解释为什么资金很重要

## 使用场景

* 撰写开源项目赞助说明
* 设计赞助层级、项目故事和影响展示
* 提升 GitHub Sponsors 页面的可信度和转化率
* 围绕 Funding、Impact、Community 等主题生成结构化结果

## 适用人群

* 开源维护者
* 开发者
* 项目运营者

## 中文 Prompt

```md
写一段话，向赞助商展示他们的资金将对我的项目和更广泛的社区产生的直接影响。
```
