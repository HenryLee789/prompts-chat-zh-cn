# 时间承诺

## 中文说明

解释为什么资金很重要

## 使用场景

* 撰写开源项目赞助说明
* 设计赞助层级、项目故事和影响展示
* 提升 GitHub Sponsors 页面的可信度和转化率
* 围绕 Funding、Time、Commitment 等主题生成结构化结果

## 适用人群

* 开源维护者
* 开发者
* 项目运营者

## 中文 Prompt

```md
解释一下赞助如何让我每周/每月投入 [X 小时/天] 的时间用于开源，比较当前的志愿者时间与潜在的赞助时间。
```
