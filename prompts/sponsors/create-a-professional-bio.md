# 创建专业简历

## 中文说明

打造引人入胜的简历

## 使用场景

* 撰写开源项目赞助说明
* 设计赞助层级、项目故事和影响展示
* 提升 GitHub Sponsors 页面的可信度和转化率
* 围绕 Bio、Profile、Introduction 等主题生成结构化结果

## 适用人群

* 开源维护者
* 开发者
* 项目运营者

## 中文 Prompt

```md
为我的个人资料撰写 GitHub 赞助商简介，重点介绍我在 [你的领域] 的经验、我的开源工作的影响以及我对社区发展的承诺。
```
