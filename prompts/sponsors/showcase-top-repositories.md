# 展示顶级存储库

## 中文说明

打造引人入胜的简历

## 使用场景

* 撰写开源项目赞助说明
* 设计赞助层级、项目故事和影响展示
* 提升 GitHub Sponsors 页面的可信度和转化率
* 围绕 Bio、Repositories、Showcase 等主题生成结构化结果

## 适用人群

* 开源维护者
* 开发者
* 项目运营者

## 中文 Prompt

```md
以激励潜在赞助商支持我的工作的方式总结我的前三个存储库（[repo1]、[repo2]、[repo3]）。
```
