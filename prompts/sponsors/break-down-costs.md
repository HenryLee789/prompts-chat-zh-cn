# 分解成本

## 中文说明

解释为什么资金很重要

## 使用场景

* 撰写开源项目赞助说明
* 设计赞助层级、项目故事和影响展示
* 提升 GitHub Sponsors 页面的可信度和转化率
* 围绕 Funding、Transparency、Costs 等主题生成结构化结果

## 适用人群

* 开源维护者
* 开发者
* 项目运营者

## 中文 Prompt

```md
为我的[项目类型]创建一个透明的明细表，说明赞助商资金的使用方式（例如，服务器成本、开发工具、会议出席情况、专用编码时间）。
```
