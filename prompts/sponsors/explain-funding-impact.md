# 解释资金影响

## 中文说明

解释为什么资金很重要

## 使用场景

* 撰写开源项目赞助说明
* 设计赞助层级、项目故事和影响展示
* 提升 GitHub Sponsors 页面的可信度和转化率
* 围绕 Funding、Impact、Sustainability 等主题生成结构化结果

## 适用人群

* 开源维护者
* 开发者
* 项目运营者

## 中文 Prompt

```md
为我的赞助商页面创建一个部分，解释资金将如何帮助我将更多时间投入到[项目/主题]、支持新贡献者并确保我的开源工作的可持续性。
```
