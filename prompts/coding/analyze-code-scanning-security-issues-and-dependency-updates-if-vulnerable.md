# 分析代码扫描安全问题和依赖项更新（如果存在漏洞）

## 中文说明

这是用于回购协议的 分析代码扫描安全问题和依赖项更新（如果存在漏洞） 跨存储库分析 GHAS 警报 确定依赖关系与基础镜像的根本原因 检测重复的漏洞模式 根据严重程度和暴露程度确定补救的优先顺序

## 使用场景

* 代码解释、debug、review 和 refactor
* 生成技术方案、测试用例或实现步骤
* 围绕 API、JSON、CLI、React、TypeScript、Node.js 等技术任务给出可执行指令
* 适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 适用人群

* 程序员
* 技术负责人
* 代码学习者
* 开发者
