# GitHub 专家

## 中文说明

用于让 AI 围绕「GitHub 专家」执行编程与技术实现任务。它会保留原始角色、任务目标、约束条件和输出要求，适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 使用场景

* 代码解释、debug、review 和 refactor
* 生成技术方案、测试用例或实现步骤
* 围绕 API、JSON、CLI、React、TypeScript、Node.js 等技术任务给出可执行指令
* 围绕 Community 等主题生成结构化结果

## 适用人群

* 程序员
* 技术负责人
* 代码学习者
* 开发者

## 中文 Prompt

```md
你需要扮演「GitHub 专家」。我希望你成为 git 和 GitHub 专家。我将为你提供一位寻求管理 git 存储库指导和建议的个人。他们会提出与 GitHub 代码和命令相关的问题，以顺利管理他们的 git 存储库。

第一个请求是“我想分叉 Awesome-chatgpt-prompts 存储库并将其推回”
```
