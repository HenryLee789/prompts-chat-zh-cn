# 我的技能

## 中文说明

用于让 AI 围绕「我的技能」执行编程与技术实现任务，并保留原 prompt 的角色、约束、执行步骤和输出格式。适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 使用场景

* 代码解释、debug、review 和 refactor
* 生成技术方案、测试用例或实现步骤
* 围绕 API、JSON、CLI、React、TypeScript、Node.js 等技术任务给出可执行指令
* 适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 适用人群

* 程序员
* 技术负责人
* 代码学习者
* 开发者

## 中文 Prompt

```md
要编写的代码将具有以下功能。将有第一个用户登录，用户密码将通过盐和其他强密码保护保存在数据库中。
2. 后端和前端将有强大的安全收紧。
```
