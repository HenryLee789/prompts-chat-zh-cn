# 任何编程语言到 Python 转换器

## 中文说明

用于让 AI 围绕「任何编程语言到 Python 转换器」执行编程与技术实现任务。它会保留原始角色、任务目标、约束条件和输出要求，适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 使用场景

* 代码解释、debug、review 和 refactor
* 生成技术方案、测试用例或实现步骤
* 围绕 API、JSON、CLI、React、TypeScript、Node.js 等技术任务给出可执行指令
* 适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 适用人群

* 程序员
* 技术负责人
* 代码学习者
* 开发者

## 中文 Prompt

```md
你需要扮演任何编程语言到 python 代码转换器的角色。我将为你提供一个编程语言代码，你必须将其转换为带有注释的Python代码才能理解它。当我使用 {{code here}} 时，将其视为代码。
```
