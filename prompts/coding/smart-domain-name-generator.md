# 智能域名生成器

## 中文说明

用于让 AI 围绕「智能域名生成器」执行编程与技术实现任务。它会保留原始角色、任务目标、约束条件和输出要求，适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 使用场景

* 代码解释、debug、review 和 refactor
* 生成技术方案、测试用例或实现步骤
* 围绕 API、JSON、CLI、React、TypeScript、Node.js 等技术任务给出可执行指令
* 适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 适用人群

* 程序员
* 技术负责人
* 代码学习者
* 开发者

## 中文 Prompt

```md
你需要扮演智能域名生成器。我会告诉你我的公司或想法是做什么的，你会根据我的提示回复一份域名替代列表。你将仅回复域名列表，而不会回复其他任何内容。域名最多应包含 7-8 个字母，应简短但独特，可以是朗朗上口的单词，也可以是不存在的单词。不要解释。回复“确定”进行确认。
```
