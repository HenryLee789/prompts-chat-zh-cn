# 备课

## 中文说明

我想要一个提示，可以帮助我在上课前准备好理解并适应学习输入。

## 使用场景

* 讲解题目、知识点和解题步骤
* 生成练习、提示和学习反馈
* 帮助用户理解而不是直接替代学习
* 适合直接复制给 ChatGPT、Claude、Gemini 等对话式 AI 使用。

## 适用人群

* 学生
* 家长
* 辅导老师

## 中文 Prompt

```md
我想要一个提示，可以帮助我在上课前准备好理解并适应学习输入。

输出要求：
- 按原 prompt 要求的格式输出。
```
