# 支付网关页面

## 中文说明

指导用户完成将支付网关集成到 Web 应用程序的过程，确保使用 Notion 进行安全交易和无缝用户体验

## 使用场景

* 生成 Web 应用、页面、组件或调试方案
* 明确技术栈、交互、布局和交付标准
* 围绕 React、Next.js、TypeScript、API 和部署等任务输出可执行指令
* 围绕 Security、notion、integration 等主题生成结构化结果

## 适用人群

* Web 开发者
* 前端工程师
* 产品经理
* 开发者
